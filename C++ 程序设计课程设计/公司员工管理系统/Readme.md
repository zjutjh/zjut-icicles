# CPCPEMS 公司员工管理系统

此项目是《C++程序设计课程设计》课程的真实提交项目，不是教师提供的示例，相关信息已做脱敏处理。

项目所有内容仅供参考，您不应该直接借用项目的任何代码作为您的课程作业，否则所有后果由您承担。

如果可用，你也可以访问项目的原始仓库 [CPCourseProjectEMS](https://github.com/MucheXD/CPCourseProjectEMS)

MucheXD 希望该项目能够对您有所帮助！

### 目录结构

- `docs` 报告中使用插图的原稿
- `report` 项目最终提交的课程报告及相关内容
- `src` 项目主要源代码
- `tools` 用于生成报告的工具，例如代码拼接、UML 自动扫描等

### 项目特点

此项目是一个基本可用的系统，满足课程设计的所有要求，详细介绍可以查看课程设计报告。

此项目具有以下较有价值的特性：

- 具有可键盘交互的控制台用户界面
- 包含一套简易事件处理与界面渲染管线
- 半解耦式模块化设计
- 使用 CMake 构建
- 使用 Git 进行版本管理，您可以查看开发历史
