## 刷课
几乎大部分课程视频都支持4倍速播放，用心的话有两节课差不多就可以刷完

## 测验
测验题目都是选择题，但不同的老师会以不同的方式布置，而且很可能标题、截止日期什么的都不一样！
但据观察题库是同一个，不管怎样题目都是一样的，所以请认真核对题目！

## 作业
作业以客观题+主观题为主，每个老师题库基本一致，但布置方式、标题、截止日期不一样！
另外，发现在某些章节会有同一道问题有的老师是选择题，有的老师是大题的情况，请自行辨别

## 期中考试
期中考试是在线、开卷 ~~（三四个AI都不一定能整出来一个正确答案）~~ 进行的，但<kbd>八周</kbd>上完课程的班级和<kbd>十六周</kbd>上完课程的班级试卷会在某些问题上不一致，所以若有同宿舍但不在同一个授课班的同学请一定注意对答案带上题目！

## 实验
实验很简单，但请认真预习、去做！预习报告字很多，但主要把操作步骤、演示视频和要记录哪些数据搞明白，这样做的时候就不盲目了！
