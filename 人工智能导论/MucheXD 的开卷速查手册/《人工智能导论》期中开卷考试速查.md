## 0. 一页总览

| 模块 | 必背关键词 | 常见问法/答案 |
|---|---|---|
| A* | $f(n)=g(n)+h(n)$；$g$ 已走代价；$h$ 启发估计 | $h=0\Rightarrow$ 统一代价搜索/Dijkstra；只看 $h\Rightarrow$ 贪婪最佳优先；$h$ 可采纳/一致时 A* 保最优 |
| 极小极大 | 零和、完全信息、MAX/MIN、效用回传 | MAX 选最大，MIN 选最小；深度受限用评价函数 |
| Alpha-Beta | $\alpha$=MAX 当前最好下界；$\beta$=MIN 当前最好上界；$\alpha\ge\beta$ 剪枝 | 剪枝不改变 minimax 结果；排序越好剪得越多 |
| MCTS | 选择-扩展-模拟-反传；UCT | 不必须人工启发函数；适合分支大、难写评估函数的博弈 |
| GA | 编码、种群、适应度、选择、交叉、变异 | 变异维持多样性、防局部最优；交叉组合基因；选择保留优良 |
| PSO | 粒子=解；速度；$pbest$；$gbest$ | 更新主要参考个体历史最优 $pbest$ 与群体全局最优 $gbest$ |
| ML 概述 | 监督/无监督/半监督/强化；训练/验证/测试 | 回归输出连续值；分类输出离散标签；聚类无标签；强化学习是序列决策 |
| 线性回归 | $h_w(x)=w^Tx+b$；MSE；最小二乘/梯度下降 | 学习率过大可能震荡/发散，过小收敛慢 |
| 逻辑回归 | Sigmoid；二分类概率；交叉熵 | 虽名“回归”，实为分类；Sigmoid 将线性输出映射到 $(0,1)$ |
| KMeans | 指定 $K$；距离；更新簇中心；SSE | 对初值/尺度/异常值敏感；偏好球状簇，不能自动发现任意形状簇 |
| SVM | 超平面；支持向量；最大间隔；核函数；软间隔 | 超平面选择标准是最大化两类最近点到超平面的间隔 |
| 决策树 | 熵、信息增益、信息增益率、Gini、剪枝 | ID3=信息增益；C4.5=信息增益率；CART 分类=Gini，回归=MSE |
| 强化学习 | Agent/Env/State/Action/Reward/Policy；$\gamma$ | $\gamma=0$ 只看即时奖励；$\gamma\to1$ 更重视长期回报 |

## 1. 启发式搜索与博弈搜索

### 1.1 A* 算法

| 项 | 内容 |
|---|---|
| 评价函数 | $f(n)=g(n)+h(n)$，选 OPEN 表中 $f$ 最小节点扩展 |
| $g(n)$ | 从起点到节点 $n$ 的实际累计代价 |
| $h(n)$ | 从 $n$ 到目标的估计代价，启发函数 |
| OPEN/CLOSED | OPEN 存待扩展节点；CLOSED 存已扩展节点 |
| 退化情形 | $h(n)=0\Rightarrow f=g$，退化为统一代价搜索/Dijkstra；若每步代价相同，可表现为 BFS；只按 $h$ 选点是贪婪最佳优先搜索 |
| 最优性条件 | $h$ 可采纳：$0\le h(n)\le h^*(n)$，不高估真实剩余代价；$h$ 一致：$h(n)\le c(n,n')+h(n')$，则首次取出目标即最优 |
| 启发函数强弱 | 在可采纳前提下，$h$ 越接近真实代价，扩展节点越少；若高估，可能更快但不保证最优 |
| 复杂度 | 最坏时间/空间指数级；主要瓶颈是 OPEN 表 |

紧凑流程：
```text
OPEN={start}; g(start)=0
while OPEN非空: 取f最小n；若n为目标返回路径；把n移入CLOSED；对每个后继m更新g、parent、f并加入/调整OPEN
```

### 1.2 极小极大搜索 Minimax

| 项 | 内容 |
|---|---|
| 适用条件 | 双人、零和、轮流行动、完全信息、确定性博弈，如井字棋、棋类 |
| 基本思想 | MAX 希望效用最大，MIN 希望效用最小；从叶节点效用向上回传 |
| 回传公式 | 若 $s$ 是 MAX 节点，$V(s)=\max_{a}V(Result(s,a))$；若是 MIN 节点，$V(s)=\min_{a}V(Result(s,a))$ |
| 截断搜索 | 树太深时设深度限制，用评价函数 $Eval(s)$ 估计非终局局面 |
| 易错点 | Minimax 假设对手理性且最优；不是贪婪选眼前最大，而是考虑对手反制后的结果 |

### 1.3 Alpha-Beta 剪枝

| 符号 | 含义 |
|---|---|
| $\alpha$ | MAX 沿当前路径已经保证能得到的最好值/下界；初始 $-\infty$ |
| $\beta$ | MIN 沿当前路径已经保证能得到的最好值/上界；初始 $+\infty$ |
| 剪枝条件 | 当 $\alpha\ge\beta$，当前分支不可能影响最终选择，停止搜索该分支 |
| 正确性 | 与完整 minimax 得到同一决策，只减少无用节点 |
| 节点排序 | 好排序：复杂度可接近 $O(b^{d/2})$；坏排序：最坏仍为 $O(b^d)$ |

紧凑伪代码：
```text
AlphaBeta(s, α, β, max?):
 if 终局/截断: return Eval(s)
 if max: v=-∞; for child: v=max(v,AB(child,α,β,false)); α=max(α,v); if α>=β: break; return v
 else:   v=+∞; for child: v=min(v,AB(child,α,β,true));  β=min(β,v); if α>=β: break; return v
```

### 1.4 蒙特卡洛树搜索 MCTS

| 阶段 | 做什么 | 关键词 |
|---|---|---|
| Selection 选择 | 从根节点按策略向下选择到可扩展节点 | UCT/UCB，平衡探索与利用 |
| Expansion 扩展 | 添加一个或多个未访问子节点 | 新动作 |
| Simulation/Rollout 模拟 | 从新节点随机或半随机走到终局，得到胜负/回报 | 不必须人工评估函数 |
| Backpropagation 反向传播 | 将结果沿路径回传，更新访问次数和平均收益 | $N,W,Q$ |

UCT 常用形式：$UCT_i=\frac{W_i}{N_i}+c\sqrt{\frac{\ln N_p}{N_i}}$。第一项利用已有胜率，第二项鼓励探索访问少的节点；$c$ 越大越探索。MCTS 适合围棋等分支巨大、评价函数难设计的问题；模拟次数越多通常越稳定，但不保证有限次必找全局最优。

## 2. 智能计算：遗传算法 GA 与粒子群 PSO

### 2.1 遗传算法 GA

site-pic: 6-1遗传算法.pdf:14

| 概念 | 对应优化问题 | 记忆 |
|---|---|---|
| 个体/染色体 | 一个候选解 | 编码后的解 |
| 基因 | 解的一个组成位/变量 | 染色体片段 |
| 种群 | 一组候选解 | 并行搜索多个区域 |
| 适应度函数 | 评价解优劣 | 决定选择概率 |
| 选择 | 优胜劣汰 | 保留/复制高适应度个体 |
| 交叉 | 重组父代基因 | 组合已有优良片段 |
| 变异 | 随机改变基因 | 增加多样性，防止局部最优 |
| 精英保留 | 直接保留最优个体 | 防止最优解丢失 |

基本流程：
```text
编码 → 初始化种群 → 计算适应度 → 选择 → 交叉 → 变异 → 新种群 → 满足终止条件则输出最优个体，否则循环
```

| 设计点 | 常考内容 |
|---|---|
| 编码 | 二进制编码、实数编码、排列编码；编码决定交叉/变异方式 |
| 初始种群 | 通常随机生成；种群太小易早熟，太大计算慢 |
| 适应度 | 最大化可直接用目标；最小化常转为 $fitness=1/(1+f)$ 或做平移 |
| 选择算子 | 轮盘赌、锦标赛、排序选择；选择压力过大会早熟 |
| 交叉算子 | 单点、多点、均匀、算术交叉；交叉概率 $p_c$ 通常较大 |
| 变异算子 | 位翻转、高斯扰动、交换变异；变异概率 $p_m$ 通常较小 |
| 终止条件 | 达最大迭代代数、适应度达到阈值、若干代无改进 |
| 缺点 | 早熟/停滞、参数敏感、收敛速度可能慢、不能保证有限代全局最优 |
| 改进 | 双种群、双倍体、自适应 $p_c/p_m$、精英策略、混合局部搜索 |

易错：选择不是“随机保留全部好个体”，而是按适应度倾向复制；交叉不是增加随机性为主，主要是重组；变异不是计算适应度，主要维持多样性。

### 2.2 粒子群优化 PSO

site-pic: 6-2群体智能.pdf:15
site-pic: 6-2群体智能.pdf:18

| 概念 | 含义 |
|---|---|
| 粒子位置 $x_i$ | 一个候选解，$n$ 维问题中 $x_i=(x_{i1},...,x_{in})$ |
| 粒子速度 $v_i$ | 解的移动方向和步长 |
| 适应度 | 用目标函数评价当前位置好坏 |
| $pbest/p_i$ | 粒子自身历史最优位置 |
| $gbest/p_g$ | 全群体历史最优位置 |
| 惯性项 | 沿原速度继续飞，保持探索 |
| 认知项 | 向自身最好经验靠近 |
| 社会项 | 向群体最好经验靠近 |

标准更新：$v_{ij}(t+1)=w v_{ij}(t)+c_1r_1(p_{ij}(t)-x_{ij}(t))+c_2r_2(p_{gj}(t)-x_{ij}(t))$，$x_{ij}(t+1)=x_{ij}(t)+v_{ij}(t+1)$。无惯性权重时相当于 $w=1$。$r_1,r_2\sim U(0,1)$ 增加随机性；$c_1,c_2$ 是学习因子/加速系数；$w$ 平衡全局探索与局部开发；常需限制 $x\in[X_{min},X_{max}]$、$v\in[V_{min},V_{max}]$。

流程：
```text
初始化粒子位置/速度 → 计算适应度 → 更新每个粒子的pbest和全局gbest → 按公式更新速度/位置 → 边界处理 → 满足终止条件输出gbest
```

| 对比 | GA | PSO |
|---|---|---|
| 灵感 | 生物进化 | 鸟群/鱼群协同觅食 |
| 个体变化 | 选择、交叉、变异产生新个体 | 粒子连续移动 |
| 信息传播 | 优秀个体通过后代传播基因 | 最优粒子吸引其他粒子 |
| 参数 | 种群规模、$p_c$、$p_m$ | $w,c_1,c_2,V_{max}$ |
| 优点 | 全局随机搜索、适合组合优化 | 公式简单、参数少、连续优化常收敛快 |
| 缺点 | 可能早熟、速度慢 | 可能早熟、易陷局部最优、依赖参数/拓扑 |

## 3. 机器学习概述

### 3.1 学习类型与任务

| 类型 | 数据 | 目标 | 例子 | 常见算法 |
|---|---|---|---|---|
| 监督学习 | 有标签 $(x,y)$ | 学 $y=f(x)$ | 房价预测、垃圾邮件分类 | 线性回归、逻辑回归、SVM、决策树 |

| 半监督学习 | 少量标签+大量无标签 | 降低标注成本 | 自训练、伪标签 | 半监督分类 |
| 强化学习 | 与环境交互得到奖励 | 学策略最大化长期回报 | 机器人走迷宫、游戏 AI | Q-learning、策略梯度 |

| 任务 | 输出 | 评价指标 |
|---|---|---|
| 回归 | 连续值 | MSE、RMSE、MAE、$R^2$ |
| 分类 | 离散类别/概率 | Accuracy、Precision、Recall、F1、AUC |
| 聚类 | 簇编号 | SSE/惯性、轮廓系数、ARI/NMI（有真标签时） |
| 降维 | 低维表示 | 保留方差、重构误差 |

### 3.2 建模流程与训练技巧

site-pic: 7 机器学习-2.1 机器学习概述.pdf:16

| 步骤 | 要点 |
|---|---|
| 问题定义 | 判断是回归、分类、聚类还是序列决策；确定指标 |
| 数据准备 | 收集、清洗缺失/异常/噪声；特征工程；标准化/归一化；划分训练/验证/测试 |
| 模型选择 | 先用简单模型做基线，再考虑复杂模型 |
| 训练调优 | 选择损失函数与优化算法；调学习率、正则化、模型复杂度 |
| 测试评估 | 只在最后用测试集估计泛化能力，避免数据泄漏 |

| 概念 | 公式/解释 |
|---|---|
| 损失函数 Loss | 单个样本误差；回归常用平方损失，分类常用交叉熵 |
| 代价函数 Cost | 全训练集平均损失，即经验风险 |
| 目标函数 Objective | 经验风险 + 正则化项/结构风险 |
| 欠拟合 | 训练误差和测试误差都高；模型太简单/特征不足 |
| 过拟合 | 训练误差低、测试误差高；模型太复杂/数据少/噪声大 |
| 正则化 | L1 促稀疏，L2 抑制大权重；可缓解过拟合 |
| 数据标准化 | 对 KMeans、SVM、梯度下降很重要；树模型相对不敏感 |

梯度下降：$w^{t+1}=w^t-\alpha\nabla J(w^t)$。$\alpha$ 为学习率：过大可能越过最低点导致震荡/发散；过小收敛慢。BGD 每步用全量样本，稳定但慢；SGD 每步用一个样本，快但波动；Mini-batch 折中且最常用。

## 4. 线性回归 Linear Regression

site-pic: 7 机器学习-2.2线性回归与逻辑回归.pdf:13

| 项 | 内容 |
|---|---|
| 任务 | 监督学习中的回归任务，预测连续值 |
| 模型 | $h_w(x)=w^Tx+b$；多特征是超平面 |
| 目标 | 找到参数使预测值与真实值误差最小 |
| 常用损失 | $J(w,b)=\frac{1}{2m}\sum_{i=1}^{m}(h_w(x^{(i)})-y^{(i)})^2$ |
| 求解 | 最小二乘/正规方程 $w=(X^TX)^{-1}X^Ty$（需可逆）；梯度下降（适合高维/大数据） |
| 梯度更新 | $w:=w-\alpha\frac{1}{m}\sum_i(h_w(x^{(i)})-y^{(i)})x^{(i)}$；$b:=b-\alpha\frac{1}{m}\sum_i(h_w(x^{(i)})-y^{(i)})$ |
| 易错 | 线性回归对异常值敏感；特征尺度差异大会影响梯度下降速度；学习率不是越大越好 |

## 5. 逻辑回归 Logistic Regression

| 项 | 内容 |
|---|---|
| 任务 | 监督学习中的分类，常用于二分类，也可扩展多分类 |
| 核心 | “线性模型 + Sigmoid”：$p=P(y=1\mid x)=\sigma(w^Tx+b)=\frac{1}{1+e^{-(w^Tx+b)}}$ |
| 决策 | 默认阈值 $0.5$：$p\ge0.5$ 判 1，否则判 0；决策边界为 $w^Tx+b=0$ |
| 损失 | 二元交叉熵 $J=-\frac{1}{m}\sum_i[y^{(i)}\log p^{(i)}+(1-y^{(i)})\log(1-p^{(i)})]$ |
| 多分类 | One-vs-Rest 训练多个二分类器；或 Softmax/多项逻辑回归 |
| 正则化 | 加 L1/L2 防过拟合，L2 常见 |
| 易错 | 名字有“回归”但本质是分类；Sigmoid 作用是把任意实数映射为 $(0,1)$ 概率；输出不是连续预测值而是概率/类别 |

逻辑回归 vs 线性回归：线性回归用 MSE 预测连续值；逻辑回归用 Sigmoid+交叉熵预测类别概率。逻辑回归的边界仍是线性的，除非手工构造非线性特征。

## 6. KMeans 聚类

| 项 | 内容 |
|---|---|
| 类型 | 无监督学习；聚类；无标签 |
| 输入 | 样本集、预先指定簇数 $K$ |
| 距离 | 常用欧氏距离，故需注意特征缩放 |
| 目标 | 最小化簇内平方和 $SSE=\sum_{k=1}^{K}\sum_{x_i\in C_k}\|x_i-\mu_k\|^2$ |
| 步骤 | 选 $K$ 个初始中心 → 分配每个样本到最近中心 → 用簇内均值更新中心 → 重复直到中心/分配不变或达迭代上限 |
| 收敛 | SSE 单调不增，有限步收敛到局部最优 |
| 优点 | 简单、快、可解释 |
| 局限 | 对 $K$、初始中心、异常值、尺度敏感；偏好球状/凸形/大小相近簇；不能自动发现环形、月牙形等任意形状簇 |
| 改进 | KMeans++ 改善初始化；多次随机初始化取最优；肘部法/轮廓系数选 $K$ |

易错：KMeans 的“中心”是均值，不一定是某个真实样本；KMeans 不是分类，因为没有标签；KMeans 不能自动决定 $K$。

## 7. 支持向量机 SVM

site-pic: 7 机器学习-2.3.1支持向量机.pdf:10

| 项 | 内容 |
|---|---|
| 任务 | 主要用于监督分类，也可做回归 SVR |
| 决策函数 | $g(x)=w^Tx+b$，$f(x)=sign(g(x))$ |
| 超平面 | $w^Tx+b=0$；$w$ 决定方向，$b$ 决定偏置 |
| 支持向量 | 离分类超平面最近、决定间隔的样本点 |
| 核心思想 | 在正确分类前提下，最大化两类支持向量到超平面的几何间隔 |
| 几何间隔 | 点到超平面距离与 $\|w\|$ 相关；最大化间隔等价于最小化 $\frac{1}{2}\|w\|^2$ |
| 硬间隔 | 数据完全线性可分：$\min \frac12\|w\|^2,\ s.t.\ y_i(w^Tx_i+b)\ge1$ |
| 软间隔 | 允许少量错误/越界：加入松弛变量 $\xi_i$ 与惩罚参数 $C$；$C$ 大更重视训练误差，$C$ 小间隔更宽更平滑 |
| 核函数 | 将低维非线性问题映射到高维线性可分；常见线性核、多项式核、RBF/高斯核、Sigmoid 核 |
| 特点 | 只由支持向量决定，解稀疏；对特征尺度敏感；小样本、高维场景表现常较好 |
| 易错 | 最大间隔不是“分类错误最少”那么简单；核函数不是显式升维计算全部坐标，而是通过内积技巧计算高维相似度 |

两点 SVM 速算：若只有一个正例 $x_+$ 和一个负例 $x_-$，最大间隔超平面是两点连线的垂直平分线。例：正例 $(-2,1)$、负例 $(1,2)$，中点 $(-\frac12,\frac32)$，法向量可取 $(3,1)$，超平面 $3(x+\frac12)+(y-\frac32)=0$，即 $3x+y=0$。

## 8. 决策树 Decision Tree

site-pic: 7 机器学习-2.4决策树.pdf:8
site-pic: 7 机器学习-2.4决策树.pdf:10

| 项 | 内容 |
|---|---|
| 结构 | 内部节点=特征测试；分支=测试结果；叶节点=类别/预测值 |
| 使用 | 从根节点按特征值走分支，到叶节点输出类别 |
| 构建 | 贪婪递归选择“最能划分数据”的特征 |
| 信息熵 | $Ent(D)=-\sum_{k=1}^{K}p_k\log_2p_k$；越小越纯；二分类 50/50 时最大为 1 |
| 条件熵 | $Ent(D\mid A)=\sum_v\frac{|D_v|}{|D|}Ent(D_v)$ |
| 信息增益 | $Gain(D,A)=Ent(D)-Ent(D\mid A)$；越大越好 |
| 信息增益率 | $GainRatio=Gain/IV(A)$，缓解信息增益偏好取值多的属性 |
| Gini | $Gini(D)=1-\sum_kp_k^2$；越小越纯 |
| 剪枝 | 预剪枝：提前停止；后剪枝：先长完整树再剪掉贡献小的子树 |
| 过拟合原因 | 分支过细，学习到噪声/孤立点 |
| 优点 | 可解释、能处理非线性、对特征尺度不敏感、可处理混合特征 |
| 缺点 | 易过拟合、对数据扰动敏感、贪婪局部最优 |

| 算法 | 特征选择 | 树形/任务 | 备注 |
|---|---|---|---|
| ID3 | 信息增益 | 多叉，分类 | 偏好取值多的属性；适合离散特征 |
| C4.5 | 信息增益率 | 多叉，分类 | ID3 改进，可处理连续值/缺失值 |
| CART | 分类用 Gini，回归用 MSE | 二叉，分类/回归 | sklearn 决策树主要实现 CART |
| 随机森林 RF | 多棵 CART 并行 bagging | 分类/回归 | 降方差、抗过拟合更强 |
| GBDT/XGBoost/LightGBM | 多棵树串行 boosting | 分类/回归 | 逐步拟合残差/梯度 |

常考计算：遇到 $0\log 0$ 按 0 处理；算信息增益时，分裂后熵要按分支样本数加权。

## 9. 强化学习 Reinforcement Learning

| 概念 | 含义 |
|---|---|
| Agent | 智能体，做决策者 |
| Environment | 环境，接收动作并返回新状态与奖励 |
| State $s$ | 环境状态 |
| Action $a$ | 智能体动作 |
| Reward $r$ | 即时奖励 |
| Policy $\pi(a\mid s)$ | 策略，状态到动作的选择规则 |
| Return $G_t$ | 累计折扣回报：$G_t=r_{t+1}+\gamma r_{t+2}+\gamma^2r_{t+3}+...$ |
| Discount $\gamma$ | 折扣因子，$0\le\gamma\le1$；越大越重视未来 |
| Value $V^\pi(s)$ | 按策略 $\pi$ 从状态 $s$ 出发的期望回报 |
| Action-Value $Q^\pi(s,a)$ | 在 $s$ 执行动作 $a$ 后再按 $\pi$ 的期望回报 |
| MDP | 马尔科夫决策过程 $(S,A,P,R,\gamma)$，当前状态包含决策所需信息 |

Bellman 思想：当前价值 = 即时奖励 + 折扣后的未来价值。最优动作价值满足 $Q^*(s,a)=\mathbb E[r+\gamma\max_{a'}Q^*(s',a')]$。Q-learning 更新：$Q(s,a)\leftarrow Q(s,a)+\alpha[r+\gamma\max_{a'}Q(s',a')-Q(s,a)]$。$\alpha$ 是学习率；方括号是 TD 误差。$\epsilon$-greedy：以 $1-\epsilon$ 选当前最优动作，以 $\epsilon$ 随机探索，平衡探索/利用。

| $\gamma$ 情形 | 行为 |
|---|---|
| $\gamma=0$ | 只关心当前即时奖励，完全短视 |
| $0<\gamma<1$ | 当前与未来都考虑，越远奖励权重越小 |
| $\gamma\to1$ | 更重视长期回报，适合长期规划，但学习更难 |

易错：强化学习不是监督学习，因为没有每一步的正确标签；奖励可能延迟；“最大化长期累计回报”不等于每一步都贪心拿最大即时奖励。

## 10. 高频易混点速判

| 问法 | 速答 |
|---|---|
| A* 中 $h=0$ 是什么？ | 统一代价搜索/Dijkstra；若边权全相同可等价 BFS |
| A* 一定最优吗？ | 不一定；需 $h$ 可采纳/一致且代价非负 |
| Alpha-Beta 剪枝是不是改变结果？ | 不改变 minimax 结果，只少搜索 |
| Alpha-Beta 何时剪？ | $\alpha\ge\beta$ |
| MCTS 是否必须人工评估函数？ | 不必须，可用 rollout 估计 |
| GA 变异目的？ | 增加种群多样性，帮助跳出局部最优 |
| PSO 两个极值？ | 个体历史最优 $pbest$ 与群体全局最优 $gbest$ |
| 监督/无监督/强化的区别？ | 有标签映射；无标签找结构；与环境交互最大化长期奖励 |
| 逻辑回归是回归吗？ | 不是，常用于分类 |
| KMeans 可以找任意形状簇吗？ | 不可以，偏好球状/凸形簇 |
| SVM 超平面标准？ | 最大化支持向量到超平面的间隔 |
| ID3/C4.5/CART 用什么？ | ID3 信息增益；C4.5 信息增益率；CART 分类 Gini、回归 MSE |
| 决策树为什么剪枝？ | 防止过拟合，提高泛化 |
| $\gamma=0$ 的 RL 智能体？ | 只看即时奖励 |
