# 《人工智能导论》期末备考指南

> 本指南用于安排复习、补足需要提前理解的题型、处理课件口径差异，并准备开卷材料。配套速查手册负责考场检索；本指南负责考前练习和应试执行。

## 1. 试卷形态与出题倾向

根据最后一课口述与教师模拟卷，试卷预计包含单选、多选、填空、计算分析、综合分析。后半部分主观题分值高，题目常把“定义/公式”嵌入具体场景，要求写出计算、设计步骤或解释原因。

| 层级 | 典型题型 | 模拟卷/口述显示的倾向 |
|---|---|---|
| 客观题 | 概念辨析、算法性质、公式符号、适用场景 | 智能体环境、BFS/DFS/UCS、SVM、逻辑有效性、预处理、PCA、关联规则、模拟退火、TF-IDF、交叉验证、RL/MDP、贝叶斯网络、AI 基础 |
| 小计算 | 距离、熵、混淆矩阵、搜索回传 | KNN 距离与投票；信息熵/条件熵/增益；Minimax 与 α-β；指标计算 |
| 综合题 | 场景设计 + 公式/步骤 + 分析改进 | RL 迷宫；GA-TSP 与 PMX；CNN 尺寸/参数/连接/FLOPs；也可能出现 A*、BP、LLM 幻觉、RDF 图 |

**答题顺序**：先快速浏览全卷和大题数据，再优先完成把握高、分值大的综合/计算题；最后回填客观题。主观题不要留空，至少写定义、公式、步骤、参数含义和合理结论。

## 2. 复习优先级

### S 级：必须手算到熟练
1. CNN：输出 H/W/C、参数、神经元、连接数、FLOPs、池化、Flatten、FC、Softmax。
2. 决策树：熵、条件熵、信息增益；ID3/C4.5/CART；剪枝与随机森林。
3. GA-TSP：排列编码、适应度、选择/交叉/变异、PMX 映射链、早熟改进。
4. 搜索：A* 的 OPEN/CLOSED 与 `f=g+h`；Minimax；从左到右 α-β 剪枝。
5. 混淆矩阵：Accuracy、Precision、Recall、F1；不平衡场景；ROC/AUC。
6. BP：全连接参数/连接/FLOPs；输出层和损失选择；梯度消失/爆炸。
7. PSO：速度/位置公式、三项含义、`w,c1,c2,r1,r2,Vmax` 的作用。

### A 级：必须能写完整简答
- 机器学习范式、训练/验证/测试、K 折、Bootstrap、过拟合/欠拟合、正则化、早停。
- 线性回归与逻辑回归；Ridge/Lasso；逻辑回归与 MLE；SVM 的支持向量、核与软间隔；SVM vs LR。
- K-means 四步、K-means++、局限、轮廓系数、适用/不适用数据；PCA 步骤。
- RL 的 MDP、Q-learning、ε-greedy、迷宫奖励设计。
- 神经元、激活函数、三种学习规则；GAN 模式崩溃；RNN/LSTM/GRU；Transformer 组件。
- LLM 三大训练阶段、幻觉原因与缓解、Prompt/微调/RAG。
- RDF 三元组拆解、知识图谱与 GNN 的关系、三大学派与 AI 伦理。

### B 级：用于防模拟卷边缘题失分
- 智能体类型和环境属性；KNN；逻辑有效性与归结；贝叶斯公式/贝叶斯网络；关联规则支持度/置信度；模拟退火；TF-IDF；产生式/框架/专家系统；CF 与模糊推理基础。

## 3. 需要提前学会的高耗时题型

### 3.1 CNN 五类数量必须分开
**提前练习原因**：参数、连接、神经元和 FLOPs 很容易混为一类，现场纠错耗时。

每题固定写表：

| 层    |       输出形状 |            参数 |  输出神经元 |         连接 |      FLOPs |
| ---- | ---------: | ------------: | -----: | ---------: | ---------: |
| Conv |     先套尺寸公式 |     核体积×核数+偏置 |  `HWC` | 局部连接数×输出体积 |    2×局部乘法数 |
| Pool | 套尺寸公式，C 不变 |          通常 0 |  `HWC` |    题干通常不要求 |    题干通常不要求 |
| FC   |       神经元数 | `(Nin+1)Nout` | `Nout` |      同参数口径 | `2NinNout` |

至少独立完成以下三题并计时：
- 64×64×3，5×5、10 核、s=1、p=2、含偏置，接 2×2 池化和 8 类 FC（模拟卷原型）。
- 14×14×3，5 个 5×5、s=1、p=1、无偏置，接池化和 4 类 FC。
- 自拟一题含 stride=2 或 dilation，检查向下取整。

### 3.2 PMX 冲突修复
**提前练习原因**：映射链方向和“中段保持不动”容易错。

练习时每题画四行：父代、中段、映射对、两个子代。区间外冲突基因沿映射连续替换，直到得到未在子代中段出现的值。务必背熟模拟卷结果：
- `P1=[1,2,3,4,5]`，`P2=[2,5,4,1,3]`，区间 2–4；
- 子代 1=`[3,5,4,1,2]`，子代 2=`[5,2,3,4,1]`。

再额外练两道含 2 次以上映射链的排列，避免只记答案。

### 3.3 α-β 剪枝画图
**提前练习原因**：概念容易背，遍历中更新范围容易混乱。

固定使用深度优先、题目指定顺序。每个节点旁记录 `(α,β)`；MAX 更新 α，MIN 更新 β；每次更新立即检查 `α≥β`。练习要求：
- 能在 5 分钟内完成三层树；
- 能解释被剪子树为何不影响根选择；
- 能比较“好排序”和“坏排序”的剪枝量；
- 能从模拟卷写出 B2 的 4、6 被剪原因。

### 3.4 A* 搜索图
**提前练习原因**：OPEN 表重排、重复状态更新和父节点回溯容易遗漏。

至少完成一题图搜索、一题八数码：逐轮写 `节点:g,h,f,parent`；同 `f` 按题目顺序；遇到更小 `g` 更新。能口头说明：`h=0→UCS`，等边权时与 BFS 的关系；可采纳与一致性的区别；曼哈顿距离通常比错位块数更强。

### 3.5 熵与混淆矩阵
- 熵题须熟练用计算器输入 `log₂x`；若计算器无直接 `log₂`，用 `log(x)/log(2)`。
- 先写比例，后写熵；条件熵一定按分支样本数加权；纯分支熵为 0。
- 混淆矩阵先圈定正类并在表边标真实/预测，防止 FP、FN 对调。练习至少覆盖：类别均衡、极度不均衡、Precision 高但 Recall 低、阈值变化与 ROC。

### 3.6 BP 与梯度问题
- 参数题统一使用 `(前层+1)×后层`；FLOPs 统一使用 `2×前层×后层`，并注明是否忽略偏置/激活。
- 提前理解链式法则的“多个局部导数相乘”，这样才能稳定回答梯度消失/爆炸原因。
- 解决措施需分类写：激活（ReLU/门控）、初始化（Xavier/He）、归一化、残差、梯度裁剪、学习率、LSTM/GRU。

### 3.7 LLM 与 RAG 综合题
用固定四段回答：定义/结构 → 三阶段训练 → 局限和幻觉原因 → Prompt、微调、RAG、工具校验与人工复核。RAG 需能写完整链路“检索—增强上下文—生成”，并指出它依赖文档质量、召回与重排，无法自动保证正确。

## 5. 课件口径差异与答题处理

| 项目 | 可能出现的口径 | 考试写法 |
|---|---|---|
| LLM 训练步骤 | 最后一课称“三个步骤”；课件细分为预训练、SFT、奖励模型、RLHF 四阶段 | 先写三大阶段“预训练—指令微调—对齐”，再注明对齐可细分为奖励模型与 RLHF |
| CNN 连接数 | 有的题把偏置也视作一条连接，有的只算输入权连接 | 先写主答案并注明“若计偏置连接则每输出单元再加 1” |
| CNN FLOPs | 乘加可算 2 FLOPs，也可能把一次 MAC 算 1 次 | 按课件常用 `2×乘法数`，并写明口径 |
| 池化参数 | 标准 Max/Avg Pool 无参数；早期 LeNet 可训练下采样层有缩放/偏置 | 依据题干；无特殊说明按 0 个可训练参数 |
| BP 输出神经元 | 26 类 One-Hot 应用 26 个输出；个别旧例写 25 个编码位 | 优先 26 个 Softmax 输出，说明“一类一神经元” |
| GRU 更新式 | 不同教材对更新门 z 的新旧状态系数互换 | 使用课件公式并说明 z 控制保留旧状态与写入新状态的比例 |
| VGG/Xception | 最后一课提到 Xception，课件重点展开 VGG、Inception、ResNet | 三者都准备：VGG 堆小核；Inception 多分支+1×1 降维；Xception 深度可分离卷积 |
| 样本数与参数数 | 课件把“浅层网络样本数最好多于参数”作为经验 | 写成课堂经验/启发，避免表述为严格定理 |
| SVM 与 LR | 口述用“数学解析方法/机器学习方法”作区分，表述较粗 | 从目标函数、损失、支持向量、概率输出、核方法五点比较，更稳妥 |

## 6. Python 资料的补充提示

用户会另行打印《D-期中 Python 速查》，期末手册未重复其中语法。D 已覆盖 Python 基础、NumPy、Pandas、Matplotlib 与常见 Scikit-learn 流程。建议另加一张“新增 API 名称索引”，无需背完整代码：

### 6.1 Scikit-learn 新增索引
- `KMeans`、`PCA`、`SVC`；`DecisionTreeClassifier`、`RandomForestClassifier`。
- `confusion_matrix`、`accuracy_score`、`precision_score`、`recall_score`、`f1_score`、`roc_curve`、`roc_auc_score`。
- `KFold`、`StratifiedKFold`、`cross_val_score`、`GridSearchCV`；预处理管线 `Pipeline`、`StandardScaler`。

### 6.2 PyTorch 最低限度索引
- 模型与层：`nn.Module`、`nn.Linear`、`nn.Conv2d`、`nn.MaxPool2d`、`nn.RNN`、`nn.LSTM`、`nn.GRU`、`nn.Dropout`、`nn.Softmax`。
- 损失与优化：`CrossEntropyLoss`、`BCELoss/BCEWithLogitsLoss`、`MSELoss`；`SGD`、`Adam`。
- 训练循环关键词：`model.train()`、`optimizer.zero_grad()`、前向、`loss.backward()`、`optimizer.step()`；推理 `model.eval()` 与 `torch.no_grad()`。
- 易错：`CrossEntropyLoss` 通常直接接 logits，无需先手工 Softmax；训练/评估模式影响 Dropout、BatchNorm。

若考试只问库名或用途，上述索引足够；若教师临时增加代码题，再打印一页最小示例。

## 7. 相对时间复习计划

### 考前 7–5 天：建立理解
- 第 1 天：搜索、Minimax/α-β、A*；完成两棵树和一题八数码。
- 第 2 天：GA/PMX、PSO；完成三道 PMX 与公式默写。
- 第 3 天：ML 指标、过拟合、线性/逻辑、SVM；完成混淆矩阵与 MLE 简答。
- 第 4 天：K-means/PCA、树/RF、RL；完成熵与迷宫设计。
- 第 5 天：神经元/BP/CNN；完成参数与 CNN 三题。
- 第 6 天：GAN、RNN/LSTM/GRU、Transformer。
- 第 7 天：LLM、知识图谱、三大学派、伦理与边缘考点。

### 考前 3–2 天：整卷训练
- 限时做教师模拟卷，不查资料；记录每次翻找耗时和错误原因。
- 第二遍允许使用速查手册，目标是每个知识点 20 秒内定位。
- 将所有错题压缩成一张“最后错误页”：公式口径、常混概念、个人计算失误。

### 考前 1 天：材料与检索
- 只复核 S 级题型和错误页，不再大规模补充资料。
- 给速查手册加实体标签：搜索、GA/PSO、ML 指标、树、RL、BP、CNN、序列模型、Transformer、LLM/KG。
- 用计算器试算 `log₂(1/3)`、指数、平方根，确认模式和电量。

## 9. 考场执行

### 9.1 开考前 3 分钟
- 浏览所有大题，圈出 CNN、熵、PMX、α-β、RL 等熟悉题。
- 在卷面边缘写下易忘公式：`f=g+h`、F1、熵、卷积输出、PSO、Q-learning。
- 估算各题时间，预留至少 10 分钟检查。

### 9.2 主观题得分结构
每题尽量包含：
1. 定义/目标；
2. 公式和符号；
3. 分步骤代入或算法过程；
4. 参数/机制解释；
5. 最终数值或明确结论；
6. 局限/改进（题目要求时）。

计算结果错时，完整公式与过程仍可能获得步骤分。简答题避免只写名词清单，应把每个关键词与作用连接起来。

### 9.3 最后检查
- 多选题确认“多选、少选、错选均不得分”的规则。
- 卷积题确认通道、偏置、向下取整、Flatten；熵题确认权重；指标题确认正类；PMX 确认无重复/缺失；α-β 确认遍历方向；Softmax 输出数等于类别数。
- 检查所有题均有内容，尤其主观题和填空题。
