### 使用说明

- 期中速查手册不含 Python 相关内容，你可能需要同时打印两份材料。
- 期末速查手册则已经包含 Python 相关内容，和独立的 Python 相关速查手册内容是一致的。
- 期末速查手册另附一份备考指南，它能够帮助你看懂速查手册并在考场快速应用。
- 《最后一课考点探查》是 Gemini 根据语音听记总结的考试重点，仅能代表 2025 级当次考试。
- Markdown 原稿中的图片链接无法使用，你可能需要从 PDF 中获取它们。

> [!important] 需要注意
>
> 这套速查手册默认使用者对人工智能（机器学习、深度学习）具有大体的理解，只提供公式、概念等需要记忆的内容。如果你对于许多内容的工作逻辑是不清楚的（例如，你不清楚神经网络的双向机制、深度学习的拟合本质等），那么你可能不适合使用这套速查手册，因为它无法在考场教会你某件事情具体是怎样运作的。

### 学习指南

人工智能是一门入门很困难的学科方向，它的学习曲线在一开始是很陡峭的，这也是为什么很多学生抱怨《人工智能导论》太难，因为仅凭课内的知识没有办法为你建构全局的认识。不过，一旦你大概理解了机器学习与深度学习的基本原理，你将能够很快的学会新的东西，因为事实上它们基本是符合直觉的。

MucheXD 认为人工智能对于计算机专业的学生还是十分重要的！如果你想从这门课中有所收获，我会建议你首先自学深度学习的相关内容，这样你会知道人工智能的运作思想，跨越入门最困难的门槛。我会推荐 _李宏毅_ 的机器学习入门课，这套课可以在 Bilibili 等平台找到副本。对于入门而言，建议的第一步是顺序学习直到学完 Transformer 的相关知识，这大概会花费你一周的课余时间（不实际做实验的情况下）。
